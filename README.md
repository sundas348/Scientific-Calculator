# 🧮 Calcite — Glassmorphism Calculator
CodeAlpha Frontend Internship · Task 2

## 📝 Description
A premium Scientific Calculator built with HTML, CSS, and JavaScript
featuring a beautiful glassmorphism design with pastel colors.

## Features
- Basic arithmetic operations (+ − × ÷)
- Scientific functions (sin, cos, tan, log, ln, √, x², x³, xʸ, 1/x, π, e, n!, |x|)
- DEG / RAD mode toggle
- Calculation history panel
- Dark mode toggle
- Keyboard support
- Smooth button animations with ripple effect
- Error handling
- Fully responsive design

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## 📁 Folder Structure
CodeAlpha_Calculator/
├── index.html
├── style.css
└── script.js

## ⌨️ Keyboard Shortcuts
- Space / K = Play Pause
- 0-9 = Numbers
- + - * / = Operators
- Enter = Equals
- Backspace = Delete
- Escape = Clear
- S = sin, C = cos, T = tan
- L = log, N = ln, Q = √
- P = π, E = e

## How to Run
1. Download or clone this repository
2. Open index.html in your browser
3. No installation or dependencies needed
